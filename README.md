# Twitch3DS

Twitch chat client for New 3DS / 2DS XL homebrew.

## Install via FBI QR Code

Open this page on your phone or PC, then scan the QR code with FBI on your 3DS.

![QR Code](https://api.qrserver.com/v1/create-qr-code/?size=200x200&data=https://github.com/gamerboyrts-a11y/twitch3ds/releases/download/v0.1.0/twitch3ds.3dsx)

Or enter the URL manually in FBI → Remote Install → Scan QR Code:

```
https://github.com/gamerboyrts-a11y/twitch3ds/releases/download/v0.1.0/twitch3ds.3dsx
```

## Features

- Twitch IRC chat (read + send)
- Device Code login (no password needed)
- Channel history
- Top screen stream info overlay
- Works on New 3DS / New 2DS XL
